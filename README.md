# 🐦 Flappy Bird — Java Edition

A fast-paced arcade game where you tap to keep a bird airborne, dodging endless pipes — simple to learn, impossible to master.

🎮 Gameplay
Guide the bird through gaps between pipes by tapping to flap. Every pipe you pass earns a point. Hit a pipe or the ground — game over.

▶️ Controls

| Key | Action |
|-----|--------|
| `SPACE` | Flap / Start / Restart |
| `↑ Arrow` | Flap / Start / Restart |

🚀 How to Run

Requirements: Java JDK 8 or higher

```bash
# Compile
javac FlappyBird.java

# Run
java FlappyBird
```
